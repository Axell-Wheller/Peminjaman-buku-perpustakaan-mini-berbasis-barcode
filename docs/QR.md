# QR Code untuk repository

Scan gambar di bawah untuk membuka repository ini di GitHub:

![Scan untuk buka repository](https://api.qrserver.com/v1/create-qr-code/?size=400x400&data=https%3A%2F%2Fgithub.com%2FAxell-Wheller%2FPeminjaman-buku-perpustakaan-mini-berbasis-barcode)

Link langsung ke file gambar (dibuat oleh API eksternal):
https://api.qrserver.com/v1/create-qr-code/?size=400x400&data=https%3A%2F%2Fgithub.com%2FAxell-Wheller%2FPeminjaman-buku-perpustakaan-mini-berbasis-barcode

Cara pakai di README.md:

```markdown
![Scan to open repository](https://api.qrserver.com/v1/create-qr-code/?size=400x400&data=https%3A%2F%2Fgithub.com%2FAxell-Wheller%2FPeminjaman-buku-perpustakaan-mini-berbasis-barcode)
```

Jika Anda ingin menyimpan file PNG langsung di repo (bukan hanya menggunakan API), beri tahu saya dan saya akan menambahkan file PNG yang di-generate ke folder `assets/`.
